# マルチアカウントガバナンス

## 概要

Organizations、SCP、CloudTrail、GuardDuty、Budgets を組み合わせたエンタープライズ規模のマルチアカウントガバナンス基盤を構築する Expert レベルの構築型問題です。

## 対象サービス

- AWS Organizations, CloudTrail, GuardDuty, Config, Budgets

## 運用要件

| 要件 | 内容 | 配点 |
|------|------|------|
| Organizations | OU 階層 | 15 点 |
| SCP | 3 種類の SCP | 25 点 |
| CloudTrail | 組織証跡 | 15 点 |
| GuardDuty | 組織脅威検出 | 15 点 |
| Config | 組織 Config | 15 点 |
| バジェット | コストアラート | 15 点 |
